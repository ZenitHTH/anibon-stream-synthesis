## ⚔️ ระบบการจัดทีมหลักและเมต้า (Core Team Archetypes)

### 1. ทีมโจมตีต่อเนื่อง (Follow-Up Attack - FuA)
*   **โครงสร้างหลัก:** Feixiao / Dr. Ratio + Topaz / March 7th (ล่าสังหาร) + Robin + Aventurine
*   **กลไก:** เน้นการออกการโจมตีซ้ำเมื่อเพื่อนร่วมทีมโจมตี เพื่อปั๊มยอดการโจมตีและเก็บสแต็คสำหรับท่าไม้ตายของเฟยเซียว

### 2. ทีมซูเปอร์เบรก (Super Break)
*   **โครงสร้างหลัก:** Firefly / Boothill + ผู้บุกเบิก (ประสาน) + Ruan Mei / Dahlia + Lingsha / Gallagher
*   **กลไก:** โจมตีเพื่อหักแถบจุดอ่อนศัตรูให้แตก จากนั้นผู้บุกเบิกประสานจะช่วยเพิ่มค่าความเสียหายพิเศษ "ซูเปอร์เบรก" ทะลุพลังป้องกันศัตรู

### 3. ทีมสายลบล้างยิงอัลติเมต (Acheron Debuff Core)
*   **โครงสร้างหลัก:** Acheron + Jiaoqiu + Pela / Silver Wolf + Aventurine / Gallagher (ใส่การ์ดเทรนด์ตลาด)
*   **กลไก:** ทุกครั้งที่ฝั่งเราสร้างสถานะผิดปกติ (Debuff) หรือแปะเครื่องหมายให้ศัตรู Acheron จะได้แต้มสำหรับสะสมใช้ท่าไม้ตาย โดยมี **Jiaoqiu** เป็นตัวปั๊มแต้มที่ดีที่สุดในเกม

### 4. ทีมสายความทรงจำอัญเชิญบริวาร (Remembrance Summon Core)
*   **โครงสร้างหลัก:** Aglaea / Castorice + ผู้บุกเบิก (ความทรงจำ) + Cyrene + Sustain
*   **กลไก:** เรียกสไปรต์ (**Memosprite**) ออกมาสู้ร่วมในสนามรบ โดยสไปรต์จะมีแถบเทิร์น ค่าสเตตัส และทักษะเฉพาะตัวที่เป็นอิสระในการสร้างดาเมจและควบคุมการเคลื่อนไหวของศัตรู

## 🗺️ ข้อมูลสถานที่สำคัญ (Xianzhou Lore & Flags)

*   **Xianzhou Yaoqing (เรือเหยาชิง / 曜青):** ยานธงทหารนำโดยนายพล **Feixiao** ที่เน้นความว่องไวในการสู้ศึก และเป็นยานเกิดของ **Jiaoqiu** (หมอทหารผู้รักการทำอาหารเปรี้ยวเผ็ด)
*   **Xianzhou Yuque (เรืออวี้เชวีย / 曜青):** ยานที่เชี่ยวชาญการหยั่งรู้ฟากฟ้าและการทำนาย นำโดยนายพล **Yao Guang**
*   **Xianzhou Luofu (เรือหลัวฝู / 罗浮):** เรือที่นำโดยนายพล **Jing Yuan** ซึ่งผู้เล่นใช้เวลาอยู่ในช่วง 1.x และ 2.x ค่อนข้างนาน

## 💬 ชื่อเล่นและศัพท์แสลงในคอมมูนิตี้ไทย (Thai Community Slang)

*   **รางดาว:** ชื่อเรียกย่อของเกม Honkai: Star Rail ในไทย
*   **พี่วันอาทิตย์ / นกยูง:** ชื่อเล่นของ **Sunday** (เนื่องจากแปลตรงตัว และลักษณะคอสตูมมีปีกนกที่หลัง)
*   **หมอพริกแกง / เจ๊งเฉียว:** ชื่อเล่นของ **Jiaoqiu** (เนื่องจากความสามารถและเนื้อเรื่องเน้นอาหารรสจัดจ้าน และการดีบัพที่ดีเลิศ)
*   **ฟูจิ:** ชื่อเล่นของ **Fugue** (ร่างพัฒนาของถิงหยุน)
*   **เจ้าหนูยิบหาย:** คำแซวดีไซน์มินิเกมและศัตรูตัวเล็กที่ชวนน่าหงุดหงิดในด่านพัซเซิล
