## ⚔️ ระบบการต่อสู้และคลาสอาชีพหลัก (Core Classes & Mechanics)

*   **Deployment Points (ค่า DP):** แต้มทรัพยากรหลักที่ใช้ในการส่งตัวละครลงสนาม ตัวละครแต่ละตัวมีค่าร่ายไม่เท่ากัน การบริหาร DP ผ่านตัวละครสาย Vanguard จึงเป็นสิ่งสำคัญในการเริ่มเกม
*   **Block Count (ค่าบล็อก):** จำนวนศัตรูที่ตัวละครนั้นสามารถยืนขวางต้านไว้ได้ หากศัตรูหลุดเกินจำนวนบล็อกจะเดินผ่านไปทันที
*   **Fast-Redeploy (การลงสนามซ้ำรวดเร็ว):** เมคคานิกเฉพาะของสาย Executor Specialist ที่ถอนตัวแล้วจะคูลดาวน์สั้นมาก (ประมาณ 18 วินาที) ทำให้สามารถส่งลงไปโจมตีสตันเป้าหมายสำคัญหรือขวางทางบอสแล้วถอนตัวได้อย่างอิสระ
*   **Elemental Damage / Decay (ความเสียหายธาตุ):** ดาเมจรูปแบบพิเศษ เช่น Necrosis (ประสาทหลอน), Burn (เผาไหม้) เมื่อเกจสะสมของศัตรูหรือฝั่งเราเต็ม จะระเบิดสร้างดาเมจจริงและลดต้านทานอย่างรุนแรง
*   **Endgame Modes (โหมดท้าทายระดับสูง):**
    *   **Integrated Strategies (IS):** โหมดแนวโร๊คไลค์ที่สุ่มไอเทม บัฟ และความท้าทาย (เช่น IS3, IS4, IS5) ผู้เล่นต้องเลือกตัวละครด้วยแต้มหวังผลจำกัด
    *   **Contingency Contract (CC):** โหมดรับคำท้าทายจำกัดขีดความสามารถ (ความเสี่ยง) เพื่อแลกกับรางวัลและคะแนนจัดอันดับ
    *   **Reclamation Algorithm (RA):** โหมดวางแผนกลยุทธ์กึ่งเอาชีวิตรอด สร้างป้อมค่ายสะสมเสบียง

## 👥 โครงสร้างทีมเมต้ายอดนิยม (Meta Team Compositions)

### 1. ทีม Wiš'adel กวาดล้างจักรวาล (Wiš'adel Hypercarry / Broken AoE Team)
*   **สมาชิก:** Wiš'adel (Main DPS) + Logos (Sub-DPS/Magic Support) + Ines (DP/Reveal) + Shu (Sustain/Warp) + Eyjafjalla the Hvit Aska (Elemental Medic)
*   **กลไก:** ใช้ **Ines** ในการปั่น DP เริ่มต้นเพื่อลง **Wiš'adel** ให้เร็วที่สุด จากนั้นใช้สกิล 3 ของเธอในการเรียกกระบอกปืนเงาสร้างความเสียหายวงกว้างทะลุทุกมิติ โดยมี **Logos** คอยเสริมความแรงดาเมจเวทและแปะสถานะ Necrosis ขณะที่ **Shu** วางโซนฮีลและบล็อกป้องกันไม่ให้ศัตรูหลุดเข้ามา

### 2. ทีมกายภาพบดขยี้ไร้ต้าน (Physical Burst Core Team)
*   **สมาชิก:** Młynar (Main Burst DPS) + Degenbrecher (AoE Slash) + Texas the Omertosa / Yato the Kirin X (Fast-Redeploy Assassins) + Bagpipe / Myrtle (DP Engine) + Saria / Medic (Sustain)
*   **กลไก:** เน้นการควบคุมระยะคูลดาวน์ท่าไม้ตายของ **Młynar** เพื่อปล่อยเพลงดาบแสงทองคำล้างหน้าจอ จากนั้นใช้ **Degenbrecher** สลับสกิลทำลายจังหวะและเคลียร์ศัตรูเวฟย่อย ส่วนตัว Fast-Redeploy จะคอยหย่อนลงไปทำดาเมจบอสเดี่ยวในจุดอับสายตา

### 3. ทีมลูปรวมธาตุตระกูลซุย (Sui Sibling Synergy Team)
*   **สมาชิก:** Shu + Yu + Chongyue + Ling
*   **กลไก:** อาศัยพรสวรรค์เฉพาะกลุ่มของตัวละครตระกูลซุย (Sui Siblings) เมื่อจัดทีมร่วมกันตั้งแต่ 3 ตัวละครขึ้นไป จะได้รับบัฟพลังชีวิต พลังโจมตี และขยายประสิทธิภาพทักษะเฉพาะสายเพิ่มขึ้นอย่างมหาศาล โดยมี **Yu** และ **Shu** เป็นเสาหลักแนวหน้า
