# FGO Knowledge & Naming Conventions

## 1. Character Naming Rules in Timestamps

### Rule A: New Character Reveals / Banner Introductions
When a **brand new Servant** is revealed, previewed, or released on a new banner:
- Format: `[Thai Transliteration] (English / 日本語)`
- Example: `อัสคาลาพอส (Ascalaphos / アスカラポス)`
- Example: `อาซูมิ อิโซระ (Azumi Isora / 阿曇磯良)`

### Rule B: Familiar Servants, Story Reading, & Gameplay Analysis
For **established/familiar Servants**, ongoing story analysis, or routine gameplay:
- Use the **familiar Thai community name/nickname** only.
- Do **NOT** append `(English / 日本語)` in parentheses.
- Example: `มาชู` (NOT `มาชู (Mash Kyrielight / マシュ)`)
- Example: `ก๊อดดอฟ` (NOT `ก๊อดดอฟ (Goredolf / ゴルドルフ)`)
